# 1、Kerberos_online_read_system

基于kerberos认证的在线阅读系统。

## 1.1 项目简介

这是一款使用kerberos进行三层认证确保报文信息安全性的在线阅读系统。

## 1.2 重构计划

- 梳理流程，搭建整体框架
- 设计交互场景，完善报文定义
- 完成 DES 和 ASE 加密模块代码编写(兼容 Linux 和 Windows)
- 完成 Linux 和 Windows 间的网络通信
- 设计高可用数据库

# 2、使用说明

服务端：Linux
用户端：Linux & Windows
数据库：MySQL

安装：
./build.sh
